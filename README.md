# dotfiles

## Prerequisites

- [yadm](https://yadm.io/docs/install) - Yet Another Dotfiles Manager
- [homebrew](https://brew.sh/) - The Package Manager for Everywhere
- [nix](https://nixos.org/download/) - Declarative builds and deployments.

## Installation

1. clone

    ```sh
    yadm clone https://github.com/taiki-kuraishi/dotfiles.git
    ```

1. install

    ```sh
    yadm bootstrap
    ```
